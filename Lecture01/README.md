# Deep Learning – Class 1  
## Introduction, Perceptron, and Logistic Regression with PyTorch

This repository contains the material for **Class 1 of the Deep Learning course**.

The class introduces the basic concepts behind neural networks and shows how a simple classifier can be implemented using **PyTorch**. The lecture is accompanied by slides, example code, and a practical exercise implemented in a Kaggle notebook.

---

## Topics Covered

This session introduces the following core concepts:

- **Introduction to Deep Learning**  
  Motivation, applications, and the role of neural networks in modern machine learning.

- **Perceptron**  
  The historical origin of neural networks and the simplest linear classification model.

- **Logistic Regression as a Neural Network**  
  Interpreting logistic regression as a **single-neuron neural network** with a sigmoid activation.

- **Binary Cross Entropy (BCE)**  
  The loss function used to train binary classifiers.

- **Introduction to PyTorch**  
  Basic components of a deep learning workflow:
  - defining models  
  - computing loss  
  - training with gradient descent

## Exercise Objective

The notebook allows students to implement a **binary classifier using PyTorch**, connecting the theoretical ideas from the lecture with a working deep learning workflow.

Students will explore:

- defining a simple model
- computing predictions
- applying binary cross entropy loss
- training a model using gradient descent

This exercise provides the **foundation for understanding more complex neural networks in later classes**.
